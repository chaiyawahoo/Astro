# How to Play:

**W/S** or **Left Stick (Vertical)**: Pitch

**A/D** or **Left Stick (Horizontal)**: Roll

**Q/E** or **Left Shoulder/Right Shoulder**: Yaw

**Alt/Shift** or **Left Trigger/Right Trigger**: Throttle
